## 잘 사용하지 않던 CSS

- margin-inline : auto
    - 질문 : 보통 가운데로 정렬할때 어떤거쓰시는지?
- text-underline-offset
    - 색상이나 굵기도 수정가능
- outline 관련 css
    - box의 너비에 잡히지않는속성
- scroll-margin-top
    - sticky 요소와 같이 사용했었음
- subgrid
    - 부모 grid를 개별 child가 상속받아서 사용가능함
- container query(vs media query)
    - viewport가 아닌 엘리먼트기준으로 css
- mask
    - 카카오톡 프로필 사진이 mask로 구현했던데
- zoom
- clip-path
- backdrop-filter
- conic-gradient
